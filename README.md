# EasyEDA Circuit AI Extension

## Overview

EasyEDA Circuit AI is an AI-powered extension for EasyEDA that enhances your circuit design workflow with intelligent assistance. This extension provides tools for generating circuit schematics using AI, importing/exporting designs, and interactive chat features to help with your electronic design projects.

## Features

- **AI Circuit Generation**: Generate circuit schematics based on natural language descriptions
- **Interactive Chat**: Chat with AI assistant for circuit design guidance and troubleshooting
- **Component Viewer**: Browse and select electronic components from LCSC database
- **Import/Export**: Import circuit designs and export BOM (Bill of Materials) in various formats including GOST standard
- **PDF Generation**: Create professional PDF documents for your designs
- **Multi-language Support**: Available in English and Chinese

## Usage

### Opening the Interface

1. Click on **CircuitAI** in the main menu
2. Select **Interface** to open the AI assistant panel

### Generating Circuits

1. Open the extension interface
2. Use the chat feature to describe your circuit requirements
3. The AI will generate a schematic based on your description
4. Review and modify the generated circuit as needed

### Exporting BOM

1. Go to **CircuitAI** > **Export** > **Elements List GOST**
2. The BOM will be generated according to GOST standards

### Component Selection

- Use the Component Viewer in the interface to browse LCSC components
- Search by part number, description, or specifications
- Add selected components to your design
